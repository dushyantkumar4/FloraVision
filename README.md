# Frontend = React + TypeScript + Tailwind

This is frontend project built on React,TypeScript & tailwind

React => frontend framwork
TypeScript => Type safety
tailwind => styling

## Setup & run 
 1. git clone https://github.com/dushyantkumar4/FloraVision.git
 2. cd FloraVision (or given project name)
 3. npm install
 4. npm run dev
   
## Feature & about 
Made it clean UI & responsive with tailwind css  
Reusable & type-safe component by using React with TypeScript
Well folder structure
